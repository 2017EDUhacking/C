# C 程式開發
>* Windows平台:Dev-c++, Visual studio 2017 community
>* Linux 平台: gcc

使用Linux 平台C程式開發:
```
1.編輯程式: gedit  myc_1_1.c
2.編譯: gcc myc_1_1.c -o myc_1_1
3.執行:./myc_1_1
```

### 說明下列程式的關鍵

```
/* myc_1_1, 我的第一個C程式碼 */ 
#include <stdio.h>
#include <stdlib.h>
int main(void)
{
   printf("Hello 偉大的恩師!\n");   	/* 印出Hello 偉大的恩師! 字串 */
   printf("Hello Taiwan!\n");        /* 印出Hello Taiwan! 字串 */   
   
   system("pause");
   return ;
   printf("Hello World!\n");
}
```
